# 别时容易见时难

基于**艾宾浩斯遗忘曲线**的智能背单词工具，支持 CET-4 全部词汇（7508 词），浏览器打开即用。

## 在线使用

**[shenzzz0628.github.io/vocab-harvester](https://shenzzz0628.github.io/vocab-harvester/)**

## 功能特色

- **艾宾浩斯记忆算法** — 7 级间隔复习（当日 → 1天 → 2天 → 4天 → 7天 → 15天 → 30天），自动安排复习计划
- **三种学习模式** — 智能模式（优先复习到期单词）、随机模式、顺序模式
- **三种学习视图** — 卡片速记、词树详情（释义/例句/短语/同反义词/真题）、列表总览
- **四种筛选** — 全部 / 待复习 / 新词 / 已掌握
- **单词搜索** — 侧边栏支持中英文搜索，点击即可跳转
- **学习统计** — 连续天数、每日目标进度条、积分等级、掌握数量
- **数据持久化** — 学习进度自动保存浏览器，刷新不丢失
- **数据导入导出** — 支持备份和恢复学习进度
- **多文件支持** — 可上传自定义 JSON 词库，切换不同词库学习

## 本地运行

需要安装 [Node.js](https://nodejs.org/)，然后：

```bash
# 克隆仓库
git clone https://github.com/shenzzz0628/vocab-harvester.git
cd vocab-harvester

# 启动服务器
node server.mjs

# 浏览器打开
# http://localhost:5500
```

或者直接双击 `start.bat`（Windows）。

## 快捷键

| 按键 | 功能 |
|------|------|
| `←` `→` | 前后翻词 |
| `空格` | 下一个 |
| `1` / `A` | 不认识 |
| `2` / `D` | 认识 |
| `3` / `S` / `F` | 显示/隐藏单词 |
| `R` | 朗读发音 |
| `Esc` | 返回卡片视图 |

## 自定义词库

拖拽 JSON 文件到页面即可上传。支持两种 JSON 格式：

1. **CET-4 格式** — 包含 `content.word.content` 结构（释义、例句、短语、同反义词、真题等）
2. **通用格式** — 包含 `word` / `translation` / `sentences` 等字段的数组

上传的文件会保存在浏览器本地（IndexedDB），下次打开自动恢复。

## 技术栈

- 纯前端单文件（HTML + CSS + JS），无框架依赖
- IndexedDB 本地文件存储
- Node.js 静态服务器（上传接口 + 动态文件列表）
- GitHub Pages 在线托管
